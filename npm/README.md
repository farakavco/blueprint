Blueprint NPM
================


## Install package

after clone project you must run this command to install development dependencies.

```bash
npm install
```

## Start dev task

start sass/coffee/es6 compilation and watch for changes.

```bash
npm run dev
```

## lint code

run sass/js lint and show result

```bash
npm run lint
```

## Screen-shot

run sass/js lint and show result

```bash
npm run screen-shot
```
